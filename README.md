
# Cyber Attack Detection in Industrial Control Systems

## Overview

This project presents a TEKAN-based Cyber Attack Detection System for Industrial Control Systems (ICS). The proposed approach combines quantum-inspired feature mapping with deep learning techniques to identify cyber attacks in industrial environments.

The model is evaluated on the WADI and SWaT datasets and demonstrates effective attack detection performance using multiple evaluation metrics.

---

## Features

- Data preprocessing and normalization
- Quantum-inspired feature encoding
- TEKAN-based deep learning architecture
- Cyber attack classification
- Performance evaluation using standard metrics
- Visualization through ROC curves, confusion matrices, and training curves

---

## Datasets

The project uses the following Industrial Control System datasets:

- WADI (Water Distribution System Dataset)
- SWaT (Secure Water Treatment Dataset)

> Note: Datasets are not included in this repository due to size and licensing restrictions.
